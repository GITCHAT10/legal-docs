# MNOS Event Protobuf v1.1

## Overview
Protobuf definitions for high-performance edge sync and offline POS scenarios.

## Parity
The `.proto` file matches `mnos_event_schema_v1_1.json` field-for-field.

## Usage
Generated classes should be used for serialization when JSON overhead is too high.
JSON remain the primary source of truth for Gateway validation.
