# UT AEROMARINE v2

UT AEROMARINE is the MNOS-controlled hybrid air–sea unmanned operations layer for the Maldives.

## Core Doctrine
- **NO PERMIT → NO MISSION**
- **NO AEGIS OPERATOR → NO LAUNCH**
- **NO REGISTERED DEVICE → NO ARMING**
- **NO AIRSPACE CLEARANCE → NO AUTONOMOUS ROUTE**
- **NO DYNAMIC HOME POINT FOR BOAT LAUNCH → NO TAKEOFF**
- **NO GEOFENCE → NO ROUTE APPROVAL**
- **NO SHADOW TRACE → NO MISSION START**
- **NO SHADOW SEAL → NO FCE BILLING**
- **NO EVENT → NO INVOICE**

## Scope
Supports Quick Response Drones, DJI Enterprise, PX4, ArduPilot (VTOL/USV/ROV), and more.
Used for Reef Survey, Resort Patrol, Emergency Response, and Marine Logistics.
